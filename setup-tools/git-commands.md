# Git Commands

Initiate git in directory \(cd to desired directory first\):

```
git init
```

Add remote repo:

```
git remote add origin https://github.com/user/repo.git
```

Stage all changes:

```
git add .
```

Commit changes:

```
git commit -m "commit message here"
```

Push changes:

```
git push origin <branchname>
```
